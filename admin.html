⚙️ Configurar Google Custom Search

Para buscar imágenes desde el admin necesitás:

  1. Ir a Google Cloud Console → Crear API Key
  2. Ir a Google CSE → Crear motor que busque en "Toda la web"
  3. Copiar el CSE ID (cx) y la API Key
💡 El CSE debe estar configurado para buscar en "toda la web" (no solo en un sitio). Sin API Key, podés usar el botón "Buscar en Google Images" que abre una pestaña aparte.
